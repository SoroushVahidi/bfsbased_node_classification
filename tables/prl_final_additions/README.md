# Supplementary PRL tables (`prl_final_additions`)

CSV files here are produced or mirrored by `scripts/prl_final_additions/build_all_prl_artifacts.py` and related tooling. They summarize **manuscript_runner / SGC v1–era** logs (e.g. win/tie/loss over 30 runs, correction behavior, threshold correlations).

**They are not the main FINAL_V3 benchmark.**

| Need | Use instead |
|------|-------------|
| Main test accuracy table (FINAL_V3, 10 splits) | `tables/main_results_prl.md` |
| Per-split log | `reports/final_method_v3_results.csv` |
| Safety / harmful splits | `reports/safety_analysis.md` |

See `reports/manuscript_repo_consistency_audit.md` for manuscript section mapping.
