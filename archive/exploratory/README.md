# Archive — Exploratory Additions

This directory contains exploratory additions that are not part of the paper's
main claim:

## `benchmark/`

A benchmark baseline suite (GPRGNN, GCNII, H2GCN, LINKX, FSGNN, ACM-GNN) plus
stubs for OrderedGNN and DJ-GNN. Archived because these baselines are not part
of the main paper comparison in FINAL_V3 and some runners are incomplete stubs.

## `external_baselines/`

A stub directory for planned vendored third-party baseline repositories. No
external code was ever added; archived as planned-but-not-executed.

See the root `archive/README.md` for details.
