# Archive — Legacy Logs

This directory contains log files from earlier development phases (UG-SGC
per-run output, old analysis CSV/MD files). These are NOT FINAL_V3 evidence.

## `ugsgc_per_split_runs/`

~270 per-split per-repeat JSON files produced by `manuscript_runner.py` during
the UG-SGC phase. Each file is a per-run record for one dataset × split ×
repeat combination. These are the raw provenance logs for the UG-SGC line.

## Other files

UG-SGC comparison runs, fairness reports, correction behavior CSVs, and old
debug text files from the original BFS-investigate experiments.

See the root `archive/README.md` for full details.
