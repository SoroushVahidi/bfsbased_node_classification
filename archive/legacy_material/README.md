# Archive — Legacy Material

This directory contains operational/provenance material that is not research
evidence:

## `run_metadata/`

Snapshots and hash files from specific cluster runs (job1, job3). Contains code
snapshots at run time and the commands used to submit those HPC jobs. Useful
only for reproducing the exact environment of historical runs.

## `broken_job1_backup/`

7 backup files from job1 (SLURM ID 883410) that was interrupted partway
through. These are copies of `manuscript_final_validation_main` files backed up
before the partial results were superseded. Not canonical evidence.

## `logs_slurm_output/`

Raw Slurm `.err` and `.out` files from PRL resubmission HPC job runs.
Operational output only; no numerical evidence.

See the root `archive/README.md` for details.
