# Canonical logs (pointer)

Final validation logs used for the PRL manuscript live in the repository **root** directory:

- `logs/manuscript_gain_over_mlp_final_validation.csv`
- `logs/manuscript_correction_behavior_final_validation.csv`
- `logs/manuscript_regime_analysis_final_validation.csv`
- `logs/comparison_runs_manuscript_final_validation.jsonl`
- `logs/comparison_summary_manuscript_final_validation.json`
- `logs/manuscript_case_studies_final_validation.md`

This folder exists so zip-based workflows have a predictable `logs/prl_final_additions/` path; it does **not** duplicate those large files.
